# 🎮 DevQuest — Трекер IT-квестов с геймификацией

Интерактивное веб-приложение для новичков в IT, где учебные задачи представлены как игровые квесты. Стек: Python, Flask, HTML, CSS, JavaScript.

## ✨ Возможности

- **10 готовых квестов** по frontend, Python, Git, автоматизации и API
- **Живой поиск** по названию квеста
- **Фильтрация по тегам** — кликни на тег, чтобы увидеть только подходящие карточки
- **Трёхэтапный прогресс** каждого квеста: Принять → В процессе → ✅ Выполнено
- **Счётчик выполненных** в шапке с анимацией
- **Сохранение прогресса** в `localStorage` — обнови страницу без потери данных
- **Адаптивный дизайн** — отлично выглядит на телефоне и компьютере

## 🖼 Интерфейс

Стиль: Clean & Playful (вдохновлён Duolingo и Notion). Светлый фон, пастельные акценты, мягкие тени, плавные анимации.

## 🚀 Запуск

```bash
pip install flask
python app.py
```

Открой `http://127.0.0.1:5000` в браузере.

## 📁 Структура

```
├── app.py              # Flask-сервер
├── content.json        # Данные квестов (8-10 штук)
├── templates/

├── static/
│   ├── style.css       # Все стили
```

## 🧩 Как добавить свой квест

Открой `content.json` и добавь новый объект в массив `quests`:

```json
{
  "id": 11,
  "title": "Название квеста",
  "description": "Описание",
  "reward": "+100 XP",
  "tags": ["python", "средне"]
}
```

Теги сложности `легко` / `средне` / `сложно` определяют цвет бейджа награды.

- **Frontend**: HTML5, CSS3 (Flexbox, Grid, CSS variables, animations)
